# JRJR Enhanced 1.2.5

## Overview

JRJR Enhanced 1.2.5 is a community-driven reconstruction of the classic Minecraft 1.2.5 modded experience inspired by JoséRobertoJR's legendary YouTube series.

Rather than simply collecting old mods, this project focuses on creating a stable, well-documented and reproducible modpack using Prism Launcher.

## Goals

* Recreate the original gameplay experience.
* Improve long-term stability.
* Eliminate block and item ID conflicts.
* Maintain complete installation documentation.
* Preserve compatibility between classic mods.
