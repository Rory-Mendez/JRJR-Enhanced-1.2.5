## Notes

Use the exact versions listed above.

Some legacy Minecraft 1.2.5 mods are no longer available from their original websites. For long-term preservation, this project references community archival sources such as Minecraft Archive whenever the official download is unavailable.

This pack uses fixed Block IDs and Item IDs. Different mod versions may introduce ID conflicts or compatibility issues.
