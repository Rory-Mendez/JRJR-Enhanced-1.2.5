# Installation Guide

## Requirements

Before installing JRJR Enhanced, make sure you have:

- Minecraft 1.2.5
- Prism Launcher (recommended)
- Java 8 (Oracle/OpenJDK)
- LWJGL 2.9.4
- Minecraft Forge 3.4.9.171

---

## Installation

1. Create a new Minecraft 1.2.5 instance.
2. Install Minecraft Forge 3.4.9.171.
3. Install the required jar mods.
4. Copy all gameplay mods into the `mods/` folder.
5. Copy the provided `config/` folder into the instance.
6. Launch the game once.
7. Verify that all mods load successfully.

---

## Recommended Launcher

Prism Launcher

The modpack was developed and tested using Prism Launcher.
