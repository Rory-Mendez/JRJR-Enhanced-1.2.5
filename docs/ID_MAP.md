# Block & Item ID Map

This document defines the reserved Block ID ranges used by JRJR Enhanced 1.2.5.

## Block ID Allocation

| Mod | Block Range | Status | Notes |
| ------ | ------------ | -------- | ------- |
| Minecraft Vanilla | 0–175 | Default | Unmodified |
| IndustrialCraft² | 190–200 | Active | Mostly default IDs |
| Forestry | 197–208, 4093–4094 | Active | Uses original IDs where possible |
| RedPower | Default | Active | No changes required |
| Railcraft | 280–286 | Active | Moved to avoid Forestry conflicts |
| Thaumcraft | 340–350 | Active | Reserved block range |
| ComputerCraft | 360–369 | Planned | Reserved |
| Portal Gun | 370–379 | Planned | Reserved |
| Steve's Carts | 380–389 | Planned | Reserved |
| Additional Pipes | 390–394 | Planned | Reserved |
| Logistics Pipes | 395–399 | Planned | Reserved |
